What is Wyndsor?

A super slick way to code responsively.

Your responsive css is just a snazzy breakpoint name away:background: (base: #FFFFFF,tablet: #2B2B2b,phone: #7825B8)

Pretty much an ongoing experiment.

We’re still percolating on what wyndsor should and can do, and your ideas and feedback can help. Join us on github and help shape wyndsor's future.

A saucier way to sass.

Wyndsor uses maps to compose dry css via the sass language. With aliases, media query consolidation, in-line automated mixins and more, wyndsor can rapidly get you from prototype to prod.

Superpowers, built-in.

Humanspeak

With wyndsor’s simple vernacular and penchant for pure media queries, writing css has never been easier or more human (thank you include-media!).

Integrations

Power up your front-end experience with integrated popular mixins, such as Animate CSS, Hover CSS, Instagram CSS, Font-Awesome, and more to come.

Augmentations

Augment your development process through media query consolidation and inline autocode, or by integrating your own mixins with wyndsor’s api.

Sass Powered

As a sass-based composer, wyndsor leverages the power of lib-sass for a highly effecient development process and flexible coding environment.

A Live Experiment

It all started as an experimental idea (when a designer decided frameworks weren’t working for him) and is being actively developed and improved.

Start composing, with wyndsor.
