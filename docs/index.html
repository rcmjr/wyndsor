Humanspeak

With wyndsor’s simple vernacular and penchant for pure media queries, writing css has never been more human.

Integrations

Power up your front-end experience with integrated popular mixins, such as Animate CSS, Hover CSS, Instagram CSS and more to come.

Augmentations

Augment your development process through media query consolidation and inline autocode, or by integrating your own mixins with wyndsor’s easy addon protocol.

Sass Powered

As a sass-based composer, wyndsor leverages the power of lib-sass for a highly effecient development process and flexible coding environment.

A Live Experiment

It all started as an experimental idea (when a designer figured frameworks weren’t working for him) and will continue to be actively developed and improved.
